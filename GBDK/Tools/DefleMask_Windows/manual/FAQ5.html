NEC PC-Engine/TurboGrafx-16 SYSTEM FAQ:


Q: I can not hear the Samples in the VGM files, what is happening?.

A: Maybe you are using an outdated VGM player, try to update it and you can hear them fine.


Q: I can not hear the LFO in some players, what is happening?.

A: Some PC-Engine emulators do not support the LFO feature, DefleMask uses internally an emulator that supports it. Check the about screen inside DefleMask for more details.
