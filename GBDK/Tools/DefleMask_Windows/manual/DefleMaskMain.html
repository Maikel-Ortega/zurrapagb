DefleMask Main



DefleMask - a Multi-System Tracker.


Supported Soundchips:

       1: YAMAHA YMU759 Mobile FM Soundchip

       2: YAMAHA YM2612 FM Soundchip

       3: Texas Instruments SN76489 PSG Soundchip

       4: Z80 Variant used by the Nintendo Game Boy

       5: Hudson Soft HuC6280

       6: Ricoh 2A03

       7: MOS Technology SID


Supported Systems:

       - YAMAHA YMU759 (Soundchip 1)

       - SEGA Genesis (Soundchip 2 + Soundchip 3)

       - SEGA Master System (Soundchip 3)

       - Nintendo Game Boy (Soundchip 4)

       - NEC PC-Engine/TurboGrafx-16 (Soundchip 5)

       - Nintendo NES (Soundchip 6)

       - Commodore 64 (Soundchip 7)



Attention:


* Please provide Administrator/Root privileges to DefleMask, it needs to to write in its directory.
* Current DefleMask character support: ASCII
