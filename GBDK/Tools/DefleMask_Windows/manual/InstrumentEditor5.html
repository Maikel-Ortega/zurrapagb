NINTENDO NES SYSTEM INSTRUMENTS EDITOR MENU:





You can control the Nintendo NES sound using STD Instruments, you can set a Volume Envelope, an Arpeggio Envelope and a Duty Cycle Envelope.
You can also load a Sample by using the "Samples" button in the instrument list, and then play them in the "PCM" channel with the designed note. Remember that you can use the Arpeggio MACRO/ENVELOPE for the PCM too.
