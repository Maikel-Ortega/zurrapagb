Commodore 64 SYSTEM FAQ:


Q: Why the volume column changes all the instruments/channels?

A: This is because the Commodore 64 do not have a per channel volume control, you have to use the ADSR to control the channel's volume, and use the volume column or the volume macro to control the global song volume.


Q: When I select the Noise waveform, the checkbox is automatically cleared, why?

A: The Noise waveform cannot be used at the same time with other Waveforms, you can mix the Triangle, Pulse and Saw, but the Noise should be selected alone, in other way it will be cleared.
