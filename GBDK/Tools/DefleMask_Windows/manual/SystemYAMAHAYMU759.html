System: YAMAHA YMU759

Soundchip: YAMAHA YMU759


Properties of the System

Note Range: C#-3 -> C-7

Max Volume: 7F

A maximum of 16 notes can be sounded, when all voices used are 2-operator.

If 4-operator voices are used the polyphony will decrease.

When all voices are 4-operators, a maximum of 8 notes can be sounded.


DefleMask can build files for YAMAHA YMU759, but newer mobile chips from YAMAHA, like the YMU762 and YMU765, have backward compatibility with the YMU759!. So you can listen songs made for the YMU759, not only on YAMAHA YMU759.

Check the compatible hardware list to find out which devices contains a YAMAHA YMU759/YMU762/YMU765/etc soundchip.
